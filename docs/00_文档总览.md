# 健身与饮食记录小程序文档总览

本文件夹包含以下文档：

1. `01_产品需求文档_PRD.md`  
   完整产品需求文档，包含产品定位、业务流程、页面结构、功能边界、异常处理、隐私合规和阶段规划。

2. `02_开发文档.md`  
   面向开发实现的技术文档，包含前后端架构、目录结构、模块职责、计算规则、图片识别实现方案和验收标准。

3. `03_Codex可执行任务清单.md`  
   可直接交给 Codex 或开发人员逐项执行的任务拆解，按初始化、用户、首页、饮食、训练、体重、数据、我的、测试分组。

4. `04_数据库表结构设计.md`  
   MySQL 表结构设计，包含用户、食物、饮食、动作、训练计划、训练执行、体重、上传文件、审计日志等核心表。

5. `05_API接口清单.md`  
   后端 API 清单，包含认证、用户、首页、食物、饮食、上传、AI 识别、训练、体重、统计等接口。

推荐阅读顺序：

```text
PRD → 开发文档 → 数据库表结构 → API接口清单 → Codex任务清单
```

推荐开发顺序：

```text
项目初始化
→ 登录与用户资料
→ 首页框架
→ 食物库与饮食记录
→ 拍照识别模拟接口
→ 训练动作库与模板
→ 训练计划
→ 训练执行与倒计时
→ 体重记录
→ 数据统计
→ 我的页面与合规
→ 联调测试
```
